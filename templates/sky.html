これはなに？

HTMLとJavaScriptとCSSで作られたプレゼンツール

HTMLなのでHTML5とかのデモにも便利

埋め込みも簡単なので資料にも最適です

機能

STYLE

各要素に「effect」を定義することで、
様々な視覚効果を演出することができます。

ThumbnailMode

スライドのサムネイルを一覧表示することができます。

  • サムネイルを表示する : "上"キー
  • サムネイルを選択する : "上下左右"キー
  • サムネイルを選択して閉じる : "Enter"キー

Auto Text ReSize

Sistereは画面のサイズに合わせて、
テキストのサイズを自動調整します。

Quick Text ReSize

実際にプロジェクターに表示した時に、小さい/大きい文字表示を簡単に切り替えることができます。
またこの機能は、全体のサイズを変更するのではなく、 文字の大きさのみを調整します.

  • 文字のサイズを大きくする : "+"キー
  • 文字のサイズを小さくする : "-"キー

使い方

ページの定義

</section>

<section>

エフェクトの付け方

CSS

[effect=delay]{
  opacity: 0;
}
[effect=delay][effected]{
  opacity: 1;
}

HTML

<section class="list">
  <h2>Agenda</h2>
  <ul>
    <li effect="delay">これはなに？</li>
    <li effect="delay">機能</li>
    <li effect="delay">使い方</li>
    <li effect="delay">まとめ</li>
  </ul>
</section>
